Instruccciones de ejecución

1. Ejecute make en la terminal
2. Ejecute el trabajo como ./trabajo archivo método, donde el archivo es alguno de los que están en la carpeta llamada "archivos" y el método puede tener como valores 1 o 0, 1 para Shannon y 0 para Huffman.

Ejemplo:
Para ejecutar el archivo dna con el algoritmo de Huffman: ./trabajo dna.100MB 0
